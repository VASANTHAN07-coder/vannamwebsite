# Vannam - Vertical Wall Painting Rover Website

This is a high-professional, responsive web application for the **Vannam** project.

## Features

- **Multilingual Support**: English, Tamil, German, Mexican Spanish.
- **Visualizer**: Upload a wall photo and visualize colors (drag & drop or click to paint).
- **Product Database**: Browse international color codes (Asian Paints, Dulux, etc.).
- **Modern Design**: Dark mode, glassmorphism, and smooth animations using Framer Motion.
- **Responsive**: Fully optimized for mobile and desktop.

## Tech Stack

- **Framework**: React (Vite)
- **Styling**: Vanilla CSS Modules + CSS Variables
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Drag & Drop**: React Dropzone

## How to Run

1.  Navigate to the project directory:
    ```bash
    cd "d:/vannam webiste"
    ```
2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the link shown in the terminal (usually `http://localhost:5173`).

## Project Structure

- `src/components`: Reusable UI components (Navbar).
- `src/pages`: Main pages (Home, Visualizer, Products, About).
- `src/context`: State management (Language).
- `src/translations.js`: Language dictionaries.
- `src/index.css`: Global styles and variables.
